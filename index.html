Chat Messenger App for Android


Brief Summary


App

For our project we have designed and implemented a simple messaging application for use on devices with the android operating system, the main criteria for the project was ‘to send data securely across a network’, to do this our app is using AES-256 encryption to encrypt the messages before sending them across the server to receiver.


Each time a message is sent through the app the user is able to specify a ‘key phrase’ themselves to encrypt the message before it is sent to the user receiving the message, who must then use this key to decrypt the message on their device.


Users must agree on a key phrase with the intended recipient which can then be entered into the input box at the top of the application screen, the sending user then types their message into the message box in the lower portion of the screen and hits the send button.


Server

The server for the app is built from the Ratchet library, see here, which is implemented in Php. The scripts involved in the server are server.php which is executed in a terminal, like powershell, and chat.php.


When the server.php code is run it creates a chat object which accepts all incoming messages and delivers them, or forwards them on, to all other connections to the server. Usually messages would only be forwarded to the intended recipient and with more time to develop the app this is something that would certainly have been implemented.
